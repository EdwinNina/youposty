## Instrucciones

1. Clonar el repositorio
2. Instalar dependencias de composer con el comando ```composer install```
3. Instalar dependencias de la vista con el comando ```npm install```
4. Levantar el servidor php con el comando ```php artisan serve```
5. Levantar la vista del proyecto con el comando ```npm run dev```
